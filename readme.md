# page-load-time

    计算页面各个指标加载所需时间

# 使用方法

```html
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>
        <style>
            html,
            body {
                margin: 0;
                padding: 0;
            }
        </style>
    </head>

    <body>
        <script src="https://cdn.jsdelivr.net/npm/page-load-time"></script>
    </body>
</html>
```
